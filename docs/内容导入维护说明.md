# 内容导入维护说明

本文记录药学院、东阳光药网站后续有更新时，如何把新闻、科研成果、学术讲座、在研管线等内容同步到本站。

以下命令均在项目根目录执行：

```bash
cd lab-web
.venv/bin/python manage.py <command>
```

如果在服务器上使用的是 conda 环境，把 `.venv/bin/python` 换成对应环境里的 `python`。

## 1. 执行前检查

导入命令会直接修改数据库和 `media/`，正式环境执行前建议先备份：

```bash
cp db.sqlite3 db.sqlite3.bak-$(date +%Y%m%d-%H%M%S)
tar -czf media.bak-$(date +%Y%m%d-%H%M%S).tar.gz media
```

导入完成后至少检查：

```bash
.venv/bin/python manage.py check
```

本地快速看页面：

```text
http://127.0.0.1:8000/news/
http://127.0.0.1:8000/research/achievements/
http://127.0.0.1:8000/research/academic-lectures/
http://127.0.0.1:8000/research/pipeline/
```

## 2. 药学院新闻与科研成果

来源：

- 药学院新闻列表：`https://sps.sysu.edu.cn/news`
- 药学院科研成果列表：`https://sps.sysu.edu.cn/research/achievements`
- 列表中跳转到微信公众号的文章也会被识别

命令：

```bash
.venv/bin/python manage.py import_sps_live_recent
```

会做的事：

- 导入 2023-01-01 之后的药学院新闻。
- 新闻写入 `新闻和通知 -> 药学院新闻`。
- 识别偏科研成果的标题，并写入 `学术科研 -> 科研成果`。
- 过滤标题或内容中包含“王红胜”的页面。
- 微信来源新闻设置为点击列表直接跳转原文。
- 尝试给新闻和科研成果抓取列表图：
  - 微信文章优先使用微信封面图。
  - 药学院官网文章使用正文第一张有效图片。
  - logo、二维码、页脚图会被跳过。

注意：

- 这个命令按标题更新已有页面；标题变化较大时可能会新增一篇，而不是覆盖旧页面。
- 有些微信页面会临时返回异常页或没有图片元数据；这种情况下图片可能抓不到，可以稍后重跑图片回填命令。
- 药学院官网有些文章正文本身没有图片，这类需要后台手动设置“列表图片”。

## 3. 药学院新闻图片回填

如果新闻已经导入，但列表页或首页仍显示占位图，可以跑：

```bash
.venv/bin/python manage.py backfill_pharmacy_news_images
```

默认只处理 `药学院新闻` 中缺少 `listing_image` 的文章，不覆盖已有图。

常用参数：

```bash
.venv/bin/python manage.py backfill_pharmacy_news_images --limit 10
.venv/bin/python manage.py backfill_pharmacy_news_images --overwrite
```

说明：

- `--limit 10`：只试跑 10 篇，适合先验证微信是否能访问。
- `--overwrite`：覆盖已有列表图，谨慎使用。

## 4. 科研成果图片回填

如果 `学术科研 -> 科研成果` 的文章缺图，可以跑：

```bash
.venv/bin/python manage.py backfill_achievement_images
```

默认只处理缺少 `listing_image` 的科研成果，不覆盖已有图。

常用参数：

```bash
.venv/bin/python manage.py backfill_achievement_images --limit 10
.venv/bin/python manage.py backfill_achievement_images --overwrite
```

注意：

- 微信来源成果通常能抓到封面图。
- 药学院官网成果如果正文没有图片，只能后台手动设置列表图片，或复用站内已有图片。

## 5. 药学院学术讲座

来源：

- `https://sps.sysu.edu.cn/event`

命令：

```bash
.venv/bin/python manage.py import_sps_events
```

会做的事：

- 导入到 `学术科研 -> 学术讲座`。
- 默认扫描最多 80 页。
- 默认只导入讲座编号约 400 讲之后的内容，更老的不导入。
- 按标题更新已有页面。

常用参数：

```bash
.venv/bin/python manage.py import_sps_events --max-pages 120
.venv/bin/python manage.py import_sps_events --min-lecture-number 500
```

如果只想要更新更近期的讲座，可以提高 `--min-lecture-number`。

## 6. 东阳光药新闻与开放课题

来源：

- 东阳光药新闻：`https://www.hecpharm.com/news_media/company_news`
- 开放课题指定文章：`https://www.hecpharm.com/news_media/company_news/1243.html`

命令：

```bash
.venv/bin/python manage.py import_hec_news
```

会做的事：

- 东阳光药普通新闻写入 `新闻和通知 -> 东阳光新闻`。
- 普通新闻只保存为跳转链接，列表点击直接打开东阳光药原文。
- 从东阳光药列表页抓取新闻封面图，写入本站列表图。
- `1243.html` 这篇开放课题文章写入 `新闻和通知 -> 开放课题`，作为本站可打开文章。
- 开放课题文章导入时会去掉正文图片，并把“东阳光药抗感染新药研发全国重点实验室”替换为“抗感染新药研发全国重点实验室”。

注意：

- 该命令会把药学院官网和微信公众号来源的新闻归到 `药学院新闻` topic。
- 东阳光新闻按来源 URL 或标题更新已有页面。

## 7. 东阳光药在研管线

来源：

- `https://www.hecpharm.com/innovating/R_D_result`

命令：

```bash
.venv/bin/python manage.py import_hec_pipeline
```

会做的事：

- 抓取东阳光药官网研发管线页面中的管线 HTML。
- 写入 `学术科研 -> 在研管线`。
- 保留原页面管线表格结构，并做了适配本站显示的字段调整。

注意：

- 这是覆盖式更新，会替换 `在研管线` 页面的正文内容。
- 如果东阳光药官网 HTML 结构改版，命令可能找不到 `.art-dev-pipeline .section-1`，需要调整脚本选择器。

## 8. 推荐更新顺序

日常同步可以按下面顺序跑：

```bash
.venv/bin/python manage.py import_sps_live_recent
.venv/bin/python manage.py import_sps_events
.venv/bin/python manage.py import_hec_news
.venv/bin/python manage.py import_hec_pipeline
.venv/bin/python manage.py backfill_pharmacy_news_images
.venv/bin/python manage.py backfill_achievement_images
.venv/bin/python manage.py check
```

如果只更新新闻，不需要跑管线；如果只更新管线，只跑 `import_hec_pipeline` 即可。

## 9. 导入后快速统计

统计药学院新闻缺图：

```bash

```

统计科研成果缺图：

```bash

```

## 10. 常见问题

### 微信文章抓不到图

可能原因：

- 微信临时返回验证页或异常页。
- 文章本身没有 `og:image`、`msg_cdn_url` 或正文图片。
- 网络请求被微信限制。

处理方式：

1. 稍后重跑对应回填命令。
2. 如果仍然没有图，在 Wagtail 后台手动给该页面设置“列表图片”。

### 页面导入了但前台看不到

检查：

- 页面是否已发布。
- 页面是否在正确父栏目下。
- topic 是否正确，例如药学院新闻应为 `pharmacy-news`。
- 浏览器是否访问了带筛选参数的 URL，例如 `/news/?topic=pharmacy-news`。

### 东阳光管线导入失败

通常是东阳光药官网 HTML 改版。检查：

```bash
.venv/bin/python manage.py import_hec_pipeline
```

如果报找不到 pipeline section，需要打开 `labsite/utils/management/commands/import_hec_pipeline.py`，调整 `fetch_pipeline_html()` 里的 CSS 选择器。

### 导入后样式没变化

如果只是跑内容导入命令，不需要重新编译前端。

只有改了这些文件才需要重新编译：

- `static_src/`
- 模板里新增大量 Tailwind class
- Sass / JS 源码

编译命令：

```bash
npm run build
```
